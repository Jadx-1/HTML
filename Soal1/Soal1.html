PointBlank adalah game RPG yang sering dimainkan oleh anak-anak zaman sekarang. Vendor game ini adalah Pt. Gemscool. Game ini mulai popular sekitar tahun 2004-2005. Game ini dibagi menjadi 2 team yaitu Counter Terorist dan Terorist. Terdapat sekitar 500 server yang disediakan oleh Gemscool. Yaitu server Newbie, Expert dan sisanya adalah server Publik. Terdapat beberapa mode permainan yaitu Deathmatch, Bomb Mission, ShotGun Match, dan yang terakhir mode Dinosaurus.

Nb: Jika anda ingin memainkan game tersebut anda harus melakukan pendaftaran di Portal Game Gemscool dan pendaftarannya gratis tidak dipungut biaya apapun.
